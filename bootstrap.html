Bootstrap

Bootstrap är ett av de mest populära CSS-ramverken. Det som är speciellt med Bootstrap är att det gör det enkelt att skapa responsiva och användarvänliga webbsidor. Bootstrap har ett stort antal färdiga komponenter och klasser för layout, typsnitt, knappar och mycket mer. Läs mer på: getbootstrap.com.

Fördelar

  • Bootstrap erbjuder färdig design för exempelvis knappar, formulär och tabeller, vilket gör att man inte behöver göra allt jobb från grunden.
  • Bootstrap gör det enkelt att skapa responsiva layouter som anpassar sig efter olika skärmstorlekar.

Nackdelar

  • Eftersom Bootstrap har ett unikt utseende kan många sidor som använder sig av Bootstrap kännas väldigt lika, speciellt om man inte gör några anpassningar av koden.
  • Bootstrap är ett stort ramverk, vilket kan göra att filstorleken på ett projekt blir onödigt stor.
