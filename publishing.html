Publisering

Jag har valt att publicera min webbplats på Netlify. Netlify är en plattform för att bygga, och publicera webbplatser och applikationer. Den är anpassad för statiska webbplatser, men kan även hantera dynamiska sidor genom exempelvis API:er. Netlify gör det enkelt att publicera med HTML, CSS och JavaScript samt andra ramverk som React och Vue.

Netlify har både en gratisversion och olika betalplaner. Gratisversionen passar bra för små projekt, inkluderar 100 GB bandbredd per månad och 300 byggförfrågningar. Den är begränsad till en användare per projekt. Betalversionerna erbjuder större bandbredd och fler funktioner.

Fördelar

  • Snabb publicering
  • Uppdateringar sker automatiskt när man pushar kod till exempelvis GitHub

Nackdelar

  • Begränsad till statiska sidor och små databaser/API:er
  • Gratisversionen har begränsad bandbredd och begränsat antal byggförfrågningar

Publisering

  • Skapa ett Git-repository på exempelvis GitHub
  • Koppla repositoryt till Netlify
  • När man gör ändringar byggs och publiceras webbplatsen automatiskt
